
# Recycle bin Managment
A Full stack Recycle Bin Managment Application


## Tech Stack

**Client:** Next JS 

**Server:** Drizzle ORM ,Neon Serverless Postgresql


## Roadmap

- Created A Next JS appliaction
   ```bash
    npx create-next-app@latest
   ```
- Added Drizzel ORM (helps to connect Js code with DB)
   ```bash
   npm i drizzle-orm pg dotenv
   npm i -D drizzle-kit tsx @types/pg
   ```
- Added Schemas to project
- Added dotenv file
 ```bash
   npm i dotenv
  ```
- Created an Account in Neon Serverless postgreSql and connected with DB.
- Schemas are pushed to  DB and we can see Schemas inside Neon Dashboard
